# 在线共享相册 — 需求与产品设计文档

> 本文档用于沉淀项目背景、产品形态、功能边界与后续讨论锚点。

---

## 1. 项目背景与动机

平时拍照后给朋友传图存在以下痛点：

- **微信传输**：自动压缩、画质损失严重，且批量发送速度慢。
- **网盘分享**：需要会员、操作繁琐、对方下载也不方便。
- **U盘 / AirDrop**：受限于物理在场或同生态系统。

因此希望自建一个 **基于 Web 的在线共享相册**，做到：

- 无需对方注册账号即可查看与下载。
- 不压缩画质（可下载原图）。
- 跨平台（浏览器、Android App、微信小程序均可访问）。
- 一次性 / 限时分享，自动清理，节省服务器资源。

---

## 2. 角色定义

| 角色 | 说明 |
| ---- | ---- |
| **分享者 (Sharer)** | 上传图片、设置有效期、获得分享码、管理自己的分享列表。 |
| **查看者 (Viewer)** | 通过分享码访问对应相册，预览缩略图、查看原图、下载（单张 / 批量）。 |
| **贡献者 (Contributor)** | 申请加入他人分享，通过审核后可协作上传照片。 |

## 3. 功能范围

### 3.1 已实现功能（2026-06 完成）

#### 账号体系
- 邮箱注册 / 登录（轻量账号）
- JWT 双 Token（access 15min + refresh 30day，rotation 防重放）
- 登录态持久化（小程序端 storage 恢复）

#### 分享创建与管理
- 创建分享：选图 → 设有效期（多档预设）→ 生成分享码 + 二维码
- 分享标题：支持自定义，创建后可点击重命名
- 图片上传：支持原图 / 微信压缩两种模式
- 补充上传：已有分享可追加新图片
- 分享管理：续期 / 提前结束 / 永久删除（ended/cleaned 状态）
- 到期自动清理（node-cron 每分钟扫描）

#### 贡献者系统
- 他人可申请加入分享
- 创建者审核通过后协作上传
- 贡献者管理弹层

#### 凭码访问
- 输入分享码进入相册
- 网格预览（缩略图，aspectFill，自适应网格）
- 大图浏览（Swiper 轮播）
- 单张原图保存到相册
- 一键存全部到相册（后台逐张下载+写入，切换页面不中断）
- 分页加载（50 张/页，加载更多）

#### 浏览体验优化
- 最近浏览记录（首页展示，自动清理已过期记录）
- 滚动位置恢复（从最近浏览进入时回到上次位置）
- 历史记录自动同步重命名

#### 性能优化
- 缩略图 WebP 格式（体积减少 25-35%）
- API gzip 压缩 + HTTP/2
- 限流保护（2000 次/分钟）

#### 多端覆盖
- **Web 前端**：Vue 3 + TDesign + UnoCSS，响应式适配移动端
- **Android App**：Expo + React Native，原生相机扫码 + 相册权限
- **微信小程序**：Taro 4 + React，与 Web 移动端 UI 对齐

#### 全局任务进度
- 上传 / 下载进度全局追踪（zustand task store）
- 浮窗进度条，任何页面可见
- 切换页面不丢失进度状态
- 完成后 3 秒自动消失

### 3.2 扩展功能（未来迭代）

- AI 筛图：自动识别模糊、废片、重复图片
- AI 选片：高光精选
- HEIC 服务端转码
- 上传分片续传 / 秒传
- 对象存储 + CDN
- Docker 部署

## 4. 关键业务流程

### 4.1 分享流程

```
分享者
  └─ 选择图片 → 写标题 → 选择有效期 → 点击创建
      └─ 服务器：
           1. 接收原图
           2. 生成预览图（缩略图 / 中等分辨率，JPEG + WebP）
           3. 生成唯一分享码 + 二维码
           4. 记录到期时间
      └─ 返回分享码给分享者 → 进入「我的分享列表」
```

### 4.2 查看流程

```
查看者
  └─ 进入 → 输入分享码（或扫码）
      └─ 服务器校验：是否存在 / 是否过期
          ├─ 有效：返回预览图列表 → 浏览缩略图 → 点击看大图 → 一键存相册
          └─ 无效：提示「分享码不存在或已过期」
```

### 4.3 生命周期管理

```
[创建] → [生效中] ──到期──→ [已过期] → [清理任务删除资源]
              │
              ├──分享者续期──→ 延长到期时间，仍处于「生效中」
              ├──分享者主动结束──→ [已手动结束]
              └──分享者永久删除──→ [硬删除，移除文件和DB记录]
```

## 5. 非功能性需求

| 维度 | 实现 |
| ---- | -------- |
| **画质** | 原图无损存储；预览图有损压缩。 |
| **存储** | 到期自动清理，支持手动永久删除。 |
| **性能** | 预览图懒加载；分页加载；WebP；gzip/HTTP2。 |
| **安全** | 分享码随机 8 位；限流防爆破；JWT 双 token rotation。 |
| **隐私** | 不做公开搜索；仅凭分享码访问；过期即删。 |
| **跨平台** | Web + Android App + 微信小程序三端覆盖。 |
