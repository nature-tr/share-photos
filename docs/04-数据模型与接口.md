# 在线共享相册 — 数据模型与 API（精简版）

> 实现侧的契约源。详细设计已在前序文档与代码中体现。

## 1. 数据库表

- **users**：`id`, `email(UNIQUE)`, `password_hash`, `display_name`, `created_at`, `updated_at`
- **refresh_tokens**：`id(jti)`, `user_id`, `token_hash`, `issued_at`, `expires_at`, `revoked_at`, `replaced_by`
- **shares**：`id`, `code(UNIQUE,8位)`, `owner_id`, `title`, `status(active|ended|cleaned)`, `created_at`, `expires_at`, `ended_at`, `cleaned_at`, `photo_count`, `total_bytes`
- **photos**：`id`, `share_id`, `original_name`, `mime_type`, `ext`, `size_bytes`, `width`, `height`, `uploaded_as(original|compressed)`, `sort_index`, `created_at`

时间字段统一存毫秒级 UNIX 时间戳。

## 2. API 列表

| 方法 | 路径 | 鉴权 | 说明 |
| --- | --- | --- | --- |
| POST | `/api/auth/register` | 否 | 注册 |
| POST | `/api/auth/login` | 否 | 登录 |
| POST | `/api/auth/refresh` | Cookie | 刷新 access token（rotate） |
| POST | `/api/auth/logout` | Cookie | 登出 |
| GET  | `/api/auth/me` | Bearer | 当前用户 |
| POST | `/api/shares` | Bearer | 创建分享 |
| GET  | `/api/shares` | Bearer | 我的分享列表 |
| GET  | `/api/shares/:id` | Bearer | 分享详情（含照片元数据） |
| PATCH | `/api/shares/:id/extend` | Bearer | 续期 |
| DELETE | `/api/shares/:id` | Bearer | 提前结束 |
| POST | `/api/shares/:id/photos` | Bearer | 上传单张照片（multipart） |
| DELETE | `/api/shares/:id/photos/:pid` | Bearer | 删除单张 |
| GET  | `/api/v/:code` | 否 | 凭码获取相册 |
| GET  | `/api/v/:code/photos/:pid/thumb` | 否 | 缩略图 |
| GET  | `/api/v/:code/photos/:pid/medium` | 否 | 中等图 |
| GET  | `/api/v/:code/photos/:pid/original` | 否 | 原图（可下载） |
| GET  | `/api/v/:code/download` | 否 | 全量 zip 下载 |

## 3. 响应格式

成功：`{ data: <T> }` ｜ 失败：`{ error: { code, message, details? } }`

## 4. 关键限制

- 单文件 50MB；单分享 ≤500 张
- 分享 TTL：5 分钟 ~ 30 天
- access token 15min；refresh token 30 天（httpOnly Cookie）
- 缩略图 JPEG q80（短边 400）；中等图 JPEG q85（长边 1600）
- 上传模式 `original | compressed`（前端选项）

## 5. 错误码（部分）

`VALIDATION_FAILED` `EMAIL_TAKEN` `INVALID_CREDENTIALS` `UNAUTHORIZED` `FORBIDDEN`
`SHARE_NOT_FOUND` `SHARE_EXPIRED` `SHARE_ENDED` `PHOTO_NOT_FOUND`
`FILE_TOO_LARGE` `UNSUPPORTED_MEDIA_TYPE` `PHOTO_LIMIT_EXCEEDED`
`REFRESH_INVALID` `REFRESH_REUSED` `RATE_LIMITED` `INTERNAL_ERROR`
